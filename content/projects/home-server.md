## Overview

I manage a robust, self-hosted Linux home server to host personal services, media, and development tools. This environment serves as my personal playground for system administration, container orchestration, and networking.

## Hardware & OS
- **Hostname**: `orion-o6`
- **Operating System**: Debian GNU/Linux 12 (bookworm)
- **Compute**: 12 CPU Cores
- **Memory**: 28.57 GB Total RAM

## Containerized Workloads
The server runs **102 total containers** with **30 actively running**, orchestrated via Docker. Some key self-hosted applications include:
- **Jellyfin** & **Jellyseerr**: Media management and streaming.
- **Immich**: High-performance self-hosted photo and video backup.
- **LibreChat**: Self-hosted AI chat interface.
- **Questarr**: Media request automation.
- **Portracker**: Internal port management and monitoring.

## System Administration & DevOps Skills
- **Resource Management**: Efficiently allocating CPU and memory across 30+ active containers.
- **Networking**: Managing complex internal port mappings and reverse proxies (exposing ports safely).
- **Maintenance**: Routine OS patching (Debian 12), Docker image updates, and persistent volume backups.
