Technical Design Decisions
#

1. Security & Authentication
#

  • Biometric Auth: The Android client uses Jetpack BiometricPrompt to secure local access.
  • JWT Authentication: The FastAPI backend issues short-lived JSON Web Tokens for session management.
  • Network Security: The Android app uses a strict Network Security Configuration, and the backend sits behind an Nginx reverse proxy with TLS termination.

2. Background Processing
#

  • WorkManager: The Android app uses Android’s WorkManager to schedule reliable background tasks that poll the backend for price alerts without draining the battery.
  • Rate Limiting: The FastAPI backend implements rate limiting middleware to prevent API abuse and throttling from external providers like Yahoo Finance.

3. Data Integrity & Code Quality
#

  • Type Safety: Pydantic models validate all incoming API payloads and outgoing responses.
  • Automated Security Scanning: Bandit and Trivy are integrated into the CI/CD pipeline to catch vulnerabilities before deployment.
  • Obfuscation: The Android APK is processed by ProGuard to shrink and obfuscate the final dex files.
